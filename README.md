# Мои тест-кейсы

## Тест-кейт №2 (позитивный). Заполнение только обязательного поля

#### Предварительные условия:
* Открыть бразуер.
* Открыть сайт по [ссылкe](https://practice-automation.com/form-fields/).

#### Шаги
* В поле Name вписать любое имя;
* Остальные поля оставить нетронутыми;
* Нажать кнопку Submit.

#### Ожидаемый результат
Появилось сообщение с текстом "Message received!", что означает успешную отправку формы.
