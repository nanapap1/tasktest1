# Мои тест-кейсы

## Тест-кейт №2 (позитивный). Заполнение только обязательного поля

#### Предварительные условия:
* Открыть бразуер.
* Открыть сайт по [ссылкe](https://practice-automation.com/form-fields/).

#### Шаги
* В поле Name вписать любое имя;
* Остальные поля оставить нетронутыми;
* Нажать кнопку Submit.

#### Ожидаемый результат
Появилось сообщение с текстом "Message received!", что означает успешную отправку формы.


## Тест-кейт №3 (негативный). Заполнение только необязательных полей

#### Предварительные условия:
* Открыть бразуер.
* Открыть сайт по [ссылкe](https://practice-automation.com/form-fields/).

#### Шаги
* В поле Password вписать любое значение;
* Из списка What is your favourite drink выбрать любые три значения;
* Из списка What is your favourite color выбрать Red;
* В поле Do you like automation? выбрать No;
* В поле Email вписать свой электронный адрес;
* В поле Message вписать Hello,world!;
* Нажать кнопку Submit.

#### Ожидаемый результат
Форма не отправилась, alert с текстом Message received! не появился.
